# IT Support Labs — Daryl Watson

## Breaking things deliberately. Fixing them methodically. Documenting it all with humor and sass.

This repository is a hands-on IT support investigation portfolio built in a 
home lab environment running Windows Server 2022, Active Directory, and 
VirtualBox. Every scenario here represents one of the most common issues 
that land on an IT support desk — reproduced deliberately, diagnosed 
systematically, and resolved with the same methodology you'd use in a 
production environment.

The writeups follow incident documentation structure. 
What you'll find in each one: what broke, what the investigation revealed, 
how it got fixed, and what it means from a security standpoint.

## Lab Environment
- Windows Server 2022 — Domain Controller 
- Windows 11 — Domain client workstation
- VirtualBox 7.1
- Wazuh SIEM — selected scenarios include detection layer
